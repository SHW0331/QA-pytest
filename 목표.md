# 1단계 : 게임 QA Automation 기본 개념
- QA 개념
    - Quality Assurance, 품질 보증
    - QA는 제품이 기획된 대로 정상적으로 작동하는지 확인하고, 버그를 최소화하는 역할
    - 테스트 대상 : PC, 모바일, 콘솔 게임
    - 테스트 환경 : 그래픽, AI, 네트워크 환경
    - 주요 이슈 : 게임 내 밸런스, 프레임 드랍, 서버 문제
    - 자동화 활용 : Airtest, Appium, 게임 내 로그 분석
- 테스트 종류
    - 기능 테스트 (Functionality Testing)
    - 호환성 테스트 (Compatibility Testing)
    - 성능 테스트 (Performance Testing)
    - 버그 탐색 (Exploratory Testing)
- 테스트 설계

- 실습
- 오픈소스 게임에서 버그를 찾아보고 정리
- 간단한 테스트 케이스 문서 작성 (JIRA, Excel)
